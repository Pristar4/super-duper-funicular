# Funktionstypen
 
 **Linearefunktion:**
$f(x)=m · x + b$
**Quadratischefunktion:**
$f(x)=ax2+bx+c$
**Potenzfunktion:**
_$f(x)=a · x^n$
**Wurzelfunktion:**
$\sqrt[n]{x} = x^\frac{1}{n}$
**Ganzrationale Funktionen und Polynome:**
$f(x) = a_nx^n + a_(n-1)x^n-1+a_1x+a_0$
**Gebrochen rationale Funktionen:**
$f(x) = \frac {p(x)}{q(x)}$
**Exponentialfunktion:**
$f(x)=a · b^x$
**[[Logarithmus]] und ln Funktion:**
$f(x) = log_b(x)$
$f(x) = ln(x)$
**Trigonometrische Funktionen:**
$f(x)=sin(x)$
$f(x)=cos(x)$
$f(x)=tan(x)$
