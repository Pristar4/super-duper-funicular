## [[Ableitungsregeln]]:
### [[Ableitung einer Konstanten]]
$f(x) = C ||f'(x)=0$
### [[Ableitung von x]]
$f(x)=x ||f'(x)=1$
### [[Potenzregel]]
$f(x) = x^n || f'(x) = n*x^{n-1}$

### [[Faktorregel]]
$f(x) = c*g(x) ||f'(x)=c*g'(x)$
### [[Summenregel]]
$f(x) = g(x) + h(x) ||f'(x) = g'(x) + h'(x)$
### [[Differenzregel]]
$f(x) = g(x) - h(x) || f'(x) = g'(x)-h'(x)$
### [[Produktregel]]
$f(x)=g(x)*h(x)||f'(x) =g'(x)*h(x) +g(x)*h'(x)$
### [[Quotientenregel]]
$f(x) =\frac {g(x)}{h(x)}||f'(x)? \frac{h(x)*g'(x)-g(x)*h'(x)}{[h(x)]^2}$

### [[Kettenregel]]
$f(x)=g(h(x))|| f'(x)=g'(h(x))*h'(x)$
